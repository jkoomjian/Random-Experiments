To make block elements share a line, use float: left. This is useful when you need to set the width/height or other block properties but also have elements next to each other.
Floated blocks are outside the normal flow, so other elements act as if they are not there. Here a blue block (100px x 100px) is inside a red box. The red box has width: 100px but no height set. In first example, the blue box pushes the red box down 100px. In second example the blue box has float: left, so the red box ignores it.
