How To Center All Elements

Inline Elements - easy just use text-align: center on the outer element
Block Level Elements With Fixed Width - Use margin: auto;
Centered Div
Block Level Elements Without Fixed Width - margin: auto won't work - instead put text-align: center; on the outer div and display: inline-block; on the inner div
Centered Div W/O Fixed Width (margin: auto)
Centered Div W/O Fixed Width (with fix)
Floated Elements - Use the following hack - right aligns w/outer elem, then left:-50% aligns r edge w/center. inner elem has left: 50% which centers it on the centeline. 

		.outer {position: relative; left: -50%; float: right;}
		.inner {position: relative; left: 50%;}
